# 💻 Terminal Portfolio — Dhruv Anand

A high-performance, developer-centric portfolio website themed as a modern terminal/IDE. Built with **React** and **Vite**, featuring a unique horizontal-scroll projects showcase and a command-line aesthetic.

![Portfolio Preview](public/terminal.svg)

## 🚀 Key Features

- **Terminal IDE Layout**: A unique UI inspired by modern code editors and terminal emulators.
- **Perfectly Responsive**:
  - **Dynamic Skills Grid**: Automatically switches from 1200px desktop views to a stable 2-column mobile grid.
  - **Horizontal Project Scroll**: A custom "page-based" horizontal scroll system that centers project cards perfectly on any screen size.
  - **Mobile-First Design**: Fully optimized for `375px` viewports with wrapping category tabs and touch-friendly controls.
- **Interactive Experience**: 
  - Integrated terminal simulation in the Hero section.
  - Animated Skill Stack with interactive command categories (`ls frontend/`, etc.).
  - Smooth matrix-style overlays and glassmorphism effects.
- **Performance Optimized**: Built on Vite with near-instant HMR and ultra-fast load times.

## 🛠️ Tech Stack

- **Frontend**: React.js, JavaScript (ES6+), Vanilla CSS
- **Design**: CSS Grid, Flexbox, Variable-based design system
- **Build Tool**: Vite
- **Deployment**: Vercel

## 📦 Project Structure

```text
src/
├── components/   # Modular UI components (Hero, Projects, Skills, etc.)
├── assets/       # Static image and media assets
├── App.jsx       # Main application routing and structure
└── index.css     # Global design system and theme variables
```

## 🛠️ Development

To run the project locally:

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
